## StackOverflow_2022_survey_exploratory_analysis
>This is an Exploratory data analysis project carried out by me using the dataset of the developers survey carried out by StackOverflow for the year 2022.
> The main analysis was carried out using the kaggle workstation for python and linked to my github repo.
> The survey CSV files where downloaded from kaggle.
> Libraries used for this analysis includes numpy, matplotlib, pandas and seaborn.
> The analysis only focuses on few areas of the survey. As time goes on, more areas will be checked.
> The project is opened for contributions and suggestions, so fee free to add in your expertise.
